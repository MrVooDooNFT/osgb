# KOCAELİ OSGB Teklif Hesaplama Sistemi

Bu proje, KOCAELİ OSGB Ahmet Akın Hatipoğlu için geliştirilmiş web tabanlı teklif oluşturma ve hesaplama aracıdır.

Sistem üzerinden:

- İş Sağlığı ve Güvenliği hizmet teklifleri
- Sağlık taraması / tahlil teklifleri

otomatik olarak oluşturulabilmektedir.

Kullanıcı tarafından girilen bilgiler doğrultusunda sistem:

- teklif hesaplamalarını yapar
- hizmet sürelerini hesaplar
- teklif önizlemesi oluşturur
- PDF çıktısı üretir

Proje tamamen web tabanlı çalışmaktadır ve GitHub Pages üzerinden yayınlanabilir yapıdadır.

## Özellikler

- Hizmet teklifi oluşturma
- Tahlil teklifi oluşturma
- Şahıs ve şirket desteği
- Otomatik hesaplama sistemi
- PDF çıktı alma
- Türkçe karakter desteği
- Teklif önizleme ekranı

## Kullanım

`index.html` dosyasını açmanız yeterlidir.

veya GitHub Pages üzerinden yayınlayabilirsiniz.
